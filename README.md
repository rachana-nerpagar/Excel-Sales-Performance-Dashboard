## 🎯 Project Objective

The objective of this project is to provide business stakeholders with an interactive dashboard to monitor:

- Sales Performance
- Profit Analysis
- Order Trends
- Warehouse Contribution
- Product Category Pricing
- Regional Order Distribution

The dashboard enables users to filter data dynamically using slicers for better decision-making.

---
## 📈 Dashboard Features

- 📌 KPI Cards
  - Total Sales
  - Total Profit
  - Total Orders
  - Average Price

- 📌 Year-wise Order Quantity Trend

- 📌 Profit Contribution by Warehouse

- 📌 Year-wise Total Sales

- 📌 Orders by Region Type

- 📌 Average Price by Product Category

- 📌 Sales vs Profit Analysis

- 📌 Sales Distribution by Sales Channel

- 📌 Interactive Slicers
  - Year
  - Product Category
  - Region Type

## 💡 Key Business Insights

- AXW291 contributes nearly **48%** of the total profit.
- Wholesale is the highest-performing sales channel.
- City region receives the maximum number of orders.
- Average product price is approximately **₹2,284**.
- Sales and profit trends can be analyzed dynamically using slicers.
